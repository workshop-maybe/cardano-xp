# 001 — Project Inception

**Date:** 2026-03-15

## What happened

Initialized the cardano-xp repo. Before writing any code, defined the five non-code outputs this project will produce:

1. **Build journal** (this directory) — A chronological record of how agent skills were used to build this project. Transparent, reproducible, tool-agnostic.
2. **Learning skills** — Claude Code skills that ship with the repo, so anyone who clones it can explore and understand the codebase interactively.
3. **XP tokenomics** — On-chain token design for the XP system (incoming).
4. **About Andamio** — Context on what Andamio is and the role it plays in enabling this kind of project.
5. **Content pipeline** — Raw material for blog posts and videos, batched at milestones.

## Decisions

- This repo is standalone and public-facing — not an internal Andamio team project.
- The process is as much the product as the code.
- The project will fork from the Andamio app template.
- Content flows to the orchestration workspace at milestones, not continuously.

## Skills used

- Conversation with Claude Code to define project scope and structure before touching code.
