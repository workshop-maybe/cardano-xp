# Skills

> **Philosophy**: Amaze and delight without overwhelming.

These are [Agent Skills](https://agentskills.io) — portable instructions that work with any compatible coding agent (Claude Code, Cursor, Copilot, Gemini CLI, Roo Code, and others).

## Core Skills (8)

| Skill | Purpose | Type |
|-------|---------|------|
| `/getting-started` | Quick win: customize theme colors in 2 minutes | Guided |
| `/auth` | API keys and JWT authentication | Reference |
| `/transactions` | TX state machine and hooks | Reference |
| `/task-lifecycle` | Hands-on: commit → submit → assess flow on preprod | Walkthrough |
| `/tx-challenge` | Learn TX by building — 4 progressive coding challenges | Challenge |
| `/design-system` | Styling patterns, components, audits | Reference |
| `/fix` | AI-assisted bug fixing | Workflow |
| `/ship` | Commit, PR, Merge workflow | Workflow |

## Learning Path

```
/getting-started  →  /auth  →  /transactions  →  /tx-challenge
     (quick win)      (concepts)   (reference)      (hands-on)
```
